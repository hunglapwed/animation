- MiniProject2 – Thiết kế Web động với Animation

 Thông tin sinh viên
- Tên sinh viên: Hoàng Ngọc Vĩnh Hưng
- Lớp: WE24-01
- Môn học: Thiết kế web động với Animation

-  Mô tả project
Website landing page cho một sản phẩm / startup công nghệ, gồm 4 trang:
- Home
- Services
- Portfolio
- Contact



-Công nghệ sử dụng
- HTML5
- CSS3 (Flexbox, Grid, Animation)
- JavaScript
- Thư viện: AOS / Swiper (nếu có)

---

 Phân chia bài LAB theo yêu cầu đề thi

  Layout & Responsive

- Header + Navigation (tất cả các trang)
- Menu sticky
- Highlight menu trang hiện tại
- Mobile menu có animation
- Layout sử dụng Flexbox và Grid

 File liên quan:
- `index.html`
- `services.html`
- `portfolio.html`
- `contact.html`
- `assets/css/styles.css`

---

 Animation giao diện
- Hero section animation (fade / slide)
- Card hover animation (scale / transform)
- Button hover animation

 Thực hiện bằng CSS transition & transform  
 File liên quan:
- `assets/css/styles.css`

---

 Scroll Animation

- Animation khi scroll vào section
- Gallery / content reveal

Sử dụng thư viện AOS  
 Gắn `data-aos="fade-up"` cho các section

 File liên quan:
- HTML các trang
- `assets/js/app.js`

---

  Form tương tác

- Trang `contact.html`

Chức năng gồm:
- Floating label bằng CSS
- Validation realtime bằng JavaScript
- Hiệu ứng loading khi submit
- Toast thông báo sau khi gửi form
- Lưu dữ liệu bằng `localStorage`

 File liên quan:
- `contact.html`
- `assets/js/app.js`
- `assets/css/styles.css`
